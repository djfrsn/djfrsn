4over
==============================================================


Build with:
 
 * [node.js](http://nodejs.org/)
 * [express.js](http://expressjs.com/)

## How to Use It

Clone this repository 

	git clone http://git.4over.com/dennisj/4over-server.git .

## Downloading dependencies

'npm install webpack -g'

#### Node clean install
'npm cache clean -f'
'npm install -g n'

#### Install local node modules
'npm install'

## Starting the server

	'npm start'

on the base folder
